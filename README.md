# ipsum-claude-marketplace

Claude marketplace plugins built for developer workflows.

## Installation

```bash
/plugin marketplace add YOUR_USERNAME/ipsum-claude-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| ipsum-example | Example plugin with a hello world skill |

## Structure

```
ipsum-claude-marketplace/
├── .claude-plugin/
│   └── marketplace.json
└── plugins/
    └── ipsum-example/
        ├── .claude-plugin/
        │   └── plugin.json
        └── skills/
            └── example-skill/
                └── SKILL.md
```
